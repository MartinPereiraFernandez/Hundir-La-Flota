# 🚢 Hundir la Flota

Proyecto desarrollado como práctica de programación, donde se implementa el clásico juego **Hundir la Flota**.  
Incluye lógica de colocación de barcos, gestión del tablero y detección de impactos.

## 🎯 Objetivos del proyecto
- Practicar programación estructurada y orientada a objetos.
- Implementar lógica de juego por turnos.
- Gestionar tableros, coordenadas y validaciones.

## 🧩 Funcionalidades
- Colocación de barcos en posiciones válidas.
- Tablero oculto para el jugador.
- Registro de disparos y casillas ya usadas.
- Detección de hundimiento de barcos.

## 🛠️ Tecnologías
- Java

## 📚 Aprendizajes
- Manejo de arrays bidimensionales.
- Validación de entradas.
- Diseño de lógica de juego.
